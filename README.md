# Dashboard de Ventas y Rentabilidad 2023-2024

## Descripción
Análisis completo de ventas y rentabilidad de una empresa comercial ficticia. 
Proyecto end-to-end que cubre desde la base de datos hasta el dashboard ejecutivo.

## Herramientas utilizadas
- **MySQL** — Base de datos y consultas de análisis
- **Power Query** — Conexión y limpieza de datos
- **Power Pivot** — Modelo de datos y medidas DAX
- **Power BI** — Dashboard ejecutivo interactivo

## Análisis realizados
- KPIs generales: Ingreso total, Costo total, Ganancia y Margen %
- Rentabilidad por categoría de producto
- Evolución mensual de ingresos 2023-2024
- Top 5 clientes por ingreso generado

## Estructura del proyecto
- `ventas.csv` — 800 transacciones 2023-2024
- `productos.csv` — Catálogo de 12 productos
- `clientes.csv` — 12 clientes con segmento y país
- `consultas.sql` — Consultas de análisis en MySQL
- `dashboard.pbix` — Dashboard en Power BI

## Resultados clave
- Ingreso total: $1.49M | Ganancia total: $666K | Margen: 44.6%
- Electrónica lidera en ingresos pero Periféricos tiene mejor margen (54%)
- Holding Lambda es el cliente más valioso con $174K en ingresos
